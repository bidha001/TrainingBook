# Training Book - Workout Diary App

## Overview
Training book is a console-based Java Application developed as part of **IDATA1003 Programming 1** at **NTNU**.

## Features

### **Level 1**
-Add a workout entry
-List all entries
-Search by date
-Delete an entry

### **Level 2**
-Trainer register
-Search by trainer name
-Search by workout title
-Search by date range
-Show trainer statistics

## How to run
1. Open the project
2. Click on WorkoutEntryBookApp
3. Click the Run button (green button)
